## Art Book Project

### Uygulamada kullanılan özellikler

- Core Data kullanımı

- Entity oluşturma

- Veri ekleme

- Veri silme

- Segue ile yine verilerin view controllerlar arası geçişi

- Uygulamaların çoğunu iPhone 11 üzerinde test ediyorum dolayısıyla diğer simülatörlerde görüntü sorunlu olabilir.
